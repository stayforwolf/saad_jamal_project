## Overview
This project is a simple interface prototype for a Library Book Borrowing System. It demonstrates key UI concepts such as clear navigation, consistent layout, and intuitive user flow.

## Implementation Approach
The system is built using **HTML and CSS**. The project includes four main pages: Home, Book Detail, My Books, and Dashboard. I inlcuded a shared navigation bar across all pages and consistent styling ensure a positive user experience.
